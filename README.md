## 📓 Installatie & Gebruik

```bash
# Cloning the repository
git clone https://github.com/JustinLung/connecting-people-realtime-web-app.git

# Moving to the folder
cd connecting-people-realtime-web-app

# Install NPM packages
npm install

# Run local server
npm run start
```
