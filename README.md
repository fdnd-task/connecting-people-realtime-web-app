## 🪢 Installatie & Gebruik
```
# Cloning the repository
git clone https://github.com/M4TThys123/SPRINT-11-Realtime-Web-App.git

# Moving to the folder
cd SPRINT-11-Realtime-Web-App

# Install NPM Dependencies
npm install

# Install NPM packages
npm run start
```
